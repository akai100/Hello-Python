## 目录

```python3
os.getcwd()        # 当前工作目录
os.chdir("/tmp")   # 切换目录

os.mkdir("a")      # 创建单级目录
os.makedirs("a/b") # 递归创建

os.rmdir("a")      # 删除空目录
os.removedirs("a/b")
```

## 文件

```python3
os.rename("a.txt", "b.txt")
os.remove("b.txt")     # 删除文件
os.stat("file.txt")   # 文件信息
```

## 目录遍历

```python3
os.listdir(".")
```

### os.walk

```python3
for root, dirs, files in os.walk("."):
    print(root, dirs, files)
```
