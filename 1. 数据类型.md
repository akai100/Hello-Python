## 1.3 列表

列表(list) 是由一系列按特定顺序排列的元素组成。

```python3
bicycles = ['trek', 'cannondale', 'redline']
```

### 1.3.1 访问列表元素

按索引访问列表元素

```python3
bicycles = ['trek', 'cannondale', 'redline']
print(bicycles[0])
```

访问列表最后一个元素的特殊语法：

```python3
print(bicycles[-1])
```

### 1.3.2 修改、添加和删除元素

1. 修改指定位置的元素：

```python3
motorcycles = ['honda', 'yamaha', 'suzuki']
motorcycles[0] = 'ducati'
```

2. 在列表末尾添加元素：

```python3
motorcycles = ['honda', 'yamaha', 'suzuki']
motorcycles.append('ducati')
```

3. 在列表中添加元素：

```python3
motorcycles = ['honda', 'yamaha', 'suzuki']
motorcycles.insert(0, 'ducati')    # 在索引0处添加元素
```

4. 使用 del 语句删除元素

```python3
motorcycles = ['honda', 'yamaha', 'suzuki']
del motorcycles[0]
```

5. 使用```pop```删除元素

```python3
motorcycles = ['honda', 'yamaha', 'suzuki']
popped_motorcycle = motorcycles.pop()
```

6. 删除列表中任意位置的元素

```python3
motorcycles = ['honda', 'yamaha', 'suzuki']

first_owned = motorcycles.pop(0)
```

7. 根据值删除元素

```python3
motorcycles = ['honda', 'yamaha', 'suzuki', 'ducati']

motrocycles.remove('ducati')
```

### 1.3.3 管理列表

1. 使用 ```sort``对列表进行永久排序

```python3
cars = ['bmw', 'audi', 'toyota', 'subaru']
cars.sort()
```

2. 使用```sorted()```函数对列表进行临时排序

```python3
cars = ['bmw', 'audi', 'toyota', 'subaru']

sorted_cars = sorted(cars)
```

3. 倒序

```python3
cars = ['bmw', 'audi', 'toyota', 'subaru']
cars.reverse()
```

4. 获取列表长度

```python3
cars = ['bmw', 'audi', 'toyota', 'subaru']
len(cars)
```

### 1.3.7 切片

```python3
players = ['charles', 'martina', 'michael', 'florence', 'eli']
print(players[0:3])

print(players[:4])

print(players[2:])

#  遍历切片

    print(player)
```

### 1.3.8 复制列表

```python3

friend_foods = my_foods[:]   # 创建副本，再将副本赋值给变量
other_foods = my_foods       # 和my_foods指向同一个列表
```

## 1.4 元组

Python 将不能修改的值称为不可变的，而不可变的列表称为**元组**。

```python3
dimensions = (200, 50)
# dimensions[0] 250 , 禁止
```

1. 遍历元组

```python3
dimensions = (200, 50)
for dimension in dimensions:
    print(dimension)
```
